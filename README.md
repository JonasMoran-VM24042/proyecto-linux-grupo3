# Proyecto Linux Grupo 3

# Descripcion del Proyecto.

# Este proyecto implementa un servidor Linux automatizado utilizando tecnologías de virtualización, 
# administración de sistemas y contenedores Docker. El objetivo principal es configurar un entorno funcional 
# dentro de una máquina virtual Linux Mint, aplicando buenas prácticas de gestión de usuarios, permisos, 
# automatización con scripts, control de versiones con Git/GitHub y despliegue de servicios web mediante contenedores.

# El proyecto incluye:

# Configuración inicial del sistema: hostname, usuarios, grupos y permisos.

# Estructura organizada de directorios para datos, scripts, web y evidencias.

# Script de monitoreo del sistema ejecutado automáticamente con cron cada 30 minutos.

# Repositorio Git local y remoto para control de versiones.

# Instalación y uso de Docker para ejecutar servicios.

# Implementación de un servidor web Nginx containerizado accesible desde el navegador.

# Documentación y soporte: Incluye guías detalladas de configuración y solución de problemas para cada componente del proyecto.
